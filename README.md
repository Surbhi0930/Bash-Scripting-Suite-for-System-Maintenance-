# Bash-Scripting-Suite-for-System-Maintenance-

A small collection of Bash scripts to automate common Linux maintenance tasks:
- `backup.sh` — create compressed backups of a directory
- `update_cleanup.sh` — update/upgrade and cleanup (apt-based systems)
- `log_monitor.sh` — monitor logs for patterns and print alerts
- `maintenance_menu.sh` — interactive menu to run scripts and schedule cron backups
